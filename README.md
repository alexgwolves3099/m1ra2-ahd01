# Proyecto m1ra2-ahd01

## Descripción del Proyecto

Ejemplos de diseño de maquetación grid y flex.

-index.html: presenta un diseño creado con Bootstrap mostrando color, estilo y modo responsivo a la pagina de inicio. A su vez enlaza cada uno de los ejercicios realizados.  

| Tecnologías |Descripción  |
|--|--|
|HTML | Se uso una estructura HTML para la formación de cada página, tomando en cuenta las buenas prácticas en el desarrollo de cada una de ellas. 
|CSS| Se uso para diseñar y agregar colores y estilo a cada pagina.
|Bootstrap| Se uso para el diseño del index.html.
